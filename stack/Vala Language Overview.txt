what is vala language
Vala is a programming language that is designed to bring modern programming language features to the GNOME development environment. It is an object-oriented language that is syntactically similar to C# and is designed to be compiled into efficient native code. Vala uses GObject, which is the object system used by GTK+, GNOME, and other libraries, as its underlying runtime, which means that it can leverage the functionality of these libraries directly.
